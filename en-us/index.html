KeyboardHolder

Remember the input method used by each app. When multiple apps are switched, it will automatically help you switch to the last input method used by the app. Eliminates the need to switch input methods after switching apps.

Download

Feature List

  • Automatically switch input method

    Automatically switch the input method specified by the App according to the currently activated App.

  • Input Method Memory

    When you use multiple apps at the same time, KeyboardHolder will remember the entry method you are currently using. When switching between different apps, KeyboardHolder will automatically set the input method used before you switch out of the app when you switch to the app.
