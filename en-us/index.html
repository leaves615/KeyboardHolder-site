KeyboardHolder

Remember the input method used by each app. When multiple apps are switched, it will automatically help you switch to the last input method used by the app. Eliminates the need to switch input methods after switching apps.

Download

input method memory

When you use multiple apps at the same time, KeyboardHolder will remember the method you are currently using. When switching between different apps, KeyboardHolder will automatically set the input method before you cut out the app when you switch to the app.

The app starts the default input method

When the app is launched for the first time, or when switching from the background to the foreground, it can automatically switch to the desired default input method. At the same time, it can be customized for different apps.
